🎓 CAED UNISINOS — Website Oficial

Website institucional do Centro Acadêmico de Direito da UNISINOS (CAED).
Projeto desenvolvido para fortalecer a presença digital, centralizar informações e facilitar a comunicação com os estudantes.

🔗 Acesse o site:
https://fabiowlademir.github.io/caed

✨ Sobre o projeto

Este projeto foi criado com o objetivo de fornecer uma plataforma moderna, responsiva e de fácil manutenção para o CAED, permitindo:

Divulgação de eventos
Centralização de informações institucionais
Facilitação do contato com os alunos
Presença digital profissional
Base para futuras automações e integrações

O site foi solicitado pela presidência do CAED como parte da estruturação digital da gestão.

🚀 Objetivos
Modernizar a comunicação do Centro Acadêmico
Criar um ponto único de acesso às informações
Garantir acessibilidade em dispositivos móveis
Permitir evolução contínua da plataforma
Servir como base para futuras integrações (formulários, inscrições, automações, etc.)
🛠️ Tecnologias utilizadas

O projeto foi desenvolvido com foco em simplicidade, performance e facilidade de manutenção.

Principais tecnologias:

HTML5
CSS3
JavaScript
Deploy via Netlify
